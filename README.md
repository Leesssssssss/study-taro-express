# YTiming小程序后端

第一次使用taro框架写小程序，本项目主要功能为倒数记日，用户可以记录某些重要事件的时间，小程序自动计算与当前时间的相差天数

# 项目运行

```bash
git clone https://github.com/Leesssssssss/study-taro-express.git

cd study-taro-express

npm install

node app.js
```
